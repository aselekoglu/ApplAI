# Approval Policy

Human approval is mandatory before any action that submits, publishes, sends, uploads, or marks an artifact/application as `ready_to_submit`.

Allowed without approval:

- score a pasted job description
- tailor a draft CV through ApplAI Core
- render DOCX/PDF draft artifacts
- save a draft job/application record

Requires approval:

- external submission
- publishing LinkedIn, GitHub, portfolio, or public content
- sending an email, message, DM, or application packet
- uploading a CV, cover letter, or application file to an external system
- changing any application or artifact status to `ready_to_submit`

When uncertain, pause and request approval.
